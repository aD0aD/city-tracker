## 贡献指南（Contributing）

感谢你愿意改进这个项目！

### 开发环境

- Node.js 18+
- npm 9+（或同等版本）

### 本地启动

```bash
npm install
npm run dev
```

### 提交规范（建议）

- **feat**: 新功能
- **fix**: 修复问题
- **docs**: 文档更新
- **refactor**: 重构（不改变功能）
- **chore**: 构建/工具相关

### 提交 PR 的流程

1. Fork 仓库
2. 新建分支：`git checkout -b feat/xxx` 或 `fix/xxx`
3. 提交代码：`git commit -m "feat: xxx"`
4. 推送分支：`git push origin feat/xxx`
5. 在 GitHub 发起 Pull Request，并描述改动原因/影响范围

### 报告问题（Issue）

请尽量提供：

- 复现步骤
- 实际结果 vs 期望结果
- 浏览器与系统信息
- 控制台报错截图/日志
